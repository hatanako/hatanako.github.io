接上一章：今天做点新功能，上一章把代码整理完了之后，我们做新功能的效率就大大增加了，现在我们来做冲刺功能：  
首先是动画部分，应该没什么好说的，这里我选用的参数是：`Trigger`，命名为`isDash` 然后来看代码这边

``` csharp
//PlayerMover.cs
public float dashSpeed = 12f;
public float dashDuration = 0.3f;
public float dashCooldown = 1f;
public float dashTimer = 0f;

private void Update()
{
    dashTimer += Time.deltaTime;
}

/// <summary>
/// 冲刺方法，允许角色在短时间内以更高的速度移动。通过调整角色的速度和持续时间来实现冲刺效果，同时确保在冲刺结束后恢复正常速度。
/// </summary>
/// <param name="transform"></param>
public void Dash(Transform transform)
{
    dashTimer = 0f;
    float dir = transform.localScale.x; // 获取当前角色的朝向
    if(dashTimer < dashDuration)
    {
        rb.velocity = new Vector2(dir * dashSpeed, 0);
    }

}

/// <summary>
/// 判断角色是否可以进行冲刺的方法，基于冲刺冷却时间和当前计时器的状态来确定角色是否能够执行冲刺动作。
/// </summary>
/// <returns></returns>
public bool CanDash()
{
    return dashTimer >= dashCooldown;
}
```

剩下的其他状态机切换过来应该不需要我写出来了，照葫芦画瓢就行了。

``` csharp
using UnityEngine;

public class PlayerDashState : BaseState
{
    private float dashTime = 0.2f;
    public override void Enter(PlayerController ctrl)
    {
        ctrl.anim.SetTrigger("Dash");
        dashTime = ctrl.move.dashDuration;
    }

    public override void Exit(PlayerController ctrl)
    {
    }

    public override void Update(PlayerController ctrl, InputManager input)
    {
        dashTime -= Time.deltaTime;
        ctrl.move.Dash(ctrl.transform);
        if (dashTime <= 0)
        {
            if (!ctrl.IsGrounded)
            {
                ctrl.ChangeState(ctrl.fallState);
            }
            else
                ctrl.ChangeState(ctrl.idleState);
        }
    }
}

```

好了，这样一个冲刺功能就做好了，是不是非常的简单，现在我们需要做下蹲，和滑铲的功能，我们甚至不需要写新的方法，通过上面写的`Dash`方法就可以实现这两个功能，但是这里为了区分开，还是独立出来了两个方法：

``` csharp
//PlayerMover.cs

public float slideSpeed = 15f;

public float slideDuration = 0.1f;
public float slideCooldown = 1f;
public float slideTimer = 0f;

 public void Slide(Transform transform,InputManager input)
 {
     slideTimer = 0f;
     GCDTimer = GCD; // 重置全局冷却时间
     float faceDir = transform.localScale.x; // 获取当前角色的朝向
     float dir = input.IsLeftMovePressed() ? -1:(input.IsRightMovePressed()?1 : faceDir); // 获取当前角色的朝向
     Flip(transform, dir);
     if (slideTimer < slideDuration)
     {
         rb.velocity = new Vector2(dir * slideSpeed, 0);
     }
 }

 public bool CanSlide()
 {
     return ((slideTimer >= slideCooldown) && (GCDTimer <= 0));
 }
```

那么新的问题就来了，我们应该如何避免冲刺和滑铲同时触发呢？这里不知道大伙有没有玩过最终幻想14，这款游戏中所有技能是存在一个公共CD的，避免你一下子把所有技能丢出去攒一波。  
而这里我们也可以用到类似的功能：GCD

``` csharp
public float GCD = 0.3f; // 全局冷却时间，防止冲刺和滑行同时触发
private float GCDTimer = 0.3f;

private void Update()
{
    dashTimer += Time.deltaTime;
    slideTimer += Time.deltaTime;
    GCDTimer -= Time.deltaTime;
}

public void Dash(Transform transform)
{
    dashTimer = 0f;
    GCDTimer = GCD; // 重置全局冷却时间
    //其他代码
}

public bool CanDash()
{
    return ((dashTimer >= dashCooldown) && (GCDTimer <= 0));
}

public void Slide(Transform transform,InputManager input)
{
    slideTimer = 0f;
    GCDTimer = GCD; // 重置全局冷却时间
    //其他代码
}

public bool CanSlide()
{
    return ((slideTimer >= slideCooldown) && (GCDTimer <= 0));
}
```
