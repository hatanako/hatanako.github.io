``` csharp
bool IsAllConnected(Dictionary<int, List<int>> graph, int roomCount)
{
    Queue<int> visiting = new Queue<int>();
    HashSet<int> visited = new HashSet<int>();
    visiting.Enqueue(0);
    visited.Add(0);
    while (visiting.Count > 0)
    {
        int node = visiting.Dequeue();
        foreach (int neighbor in graph[node])
        {
            if (!visited.Contains(neighbor))
            {
                visited.Add(neighbor);
                visiting.Enqueue(neighbor);
            }
        }
    }
    return visited.Count == roomCount;
}
```

### 现在开始：递归  
递归只有一个核心思想：一个函数在自己的函数体里调用自己。  
听起来很绕，但其实你每天都在用这个思维，只是没意识到。  
比如"计算 1+2+3+...+n 的总和"，你可以这样想：  
sum(n) = n + sum(n-1)  
sum(n-1) = (n-1) + sum(n-2)  
sum(n-2) = (n-2) + sum(n-3)  
...  
sum(1) = 1   ← 到这里不再往下，这叫"终止条件"  
每一层都把问题缩小一点，直到小到能直接回答为止。  
用代码写出来就是：  
``` csharp
int Sum(int n)
{
    if (n == 1) return 1;        // 终止条件：最小的情况直接给答案
    return n + Sum(n - 1);       // 否则：当前值 + 更小问题的答案
}
```

递归函数永远只有这两部分：**终止条件** 和 **缩小问题**。

归并排序是冒泡排序完全不同的思路，它用的是分治——把大问题切成两半，分别解决，再把结果合并。  
```
[5, 3, 8, 1, 9, 2]
        ↓ 切成两半
[5, 3, 8]    [1, 9, 2]
    ↓              ↓
[5,3] [8]    [1,9] [2]
  ↓    ↓       ↓    ↓
[5][3] [8]   [1][9] [2]
  ↓              ↓
合并[3,5]      合并[1,9]
    ↓              ↓
合并[3,5,8]    合并[1,2,9]
        ↓ 最后合并
   [1,2,3,5,8,9]
```
难点不在"切"，切很简单，每次从中间切开就行。  
难点在合并：两个已经排好序的数组，怎么合并成一个排好序的数组？  
比如把 [3, 5, 8] 和 [1, 2, 9] 合并成 [1, 2, 3, 5, 8, 9]。  
思路是：两个数组各放一个指针，每次比较两个指针指向的数，把小的那个取出来放进结果，然后那边的指针往右移一步，直到某一边取完为止，把另一边剩下的全部接到结果后面。  
先只做合并这一步——  
```
题目：
写一个函数：
int[] Merge(int[] left, int[] right)

输入两个已经排好序的数组，返回合并后的排好序的数组。

测试：
left  = [3, 5, 8]
right = [1, 2, 9]
结果应该是 [1, 2, 3, 5, 8, 9]
```

``` csharp
int[] Merge(int[] left, int[] right)
{
    int[] result = new int[left.Length + right.Length];
    int i = 0, j = 0, index = 0;

    // 两边都还有元素时，比较取小的
    while (i < left.Length && j < right.Length)
    {
        if (left[i] <= right[j])
            result[index++] = left[i++];
        else
            result[index++] = right[j++];
    }

    // 左边还有剩
    while (i < left.Length)
        result[index++] = left[i++];

    // 右边还有剩
    while (j < right.Length)
        result[index++] = right[j++];

    return result;
}
```
