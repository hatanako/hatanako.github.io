## Console：

> console.WriteLine("");意义为在控制台输出一行文字，在输出完毕后进入下一行。
> 
> Console.Write("");不同于WriteLine，它仅仅只是输出一行文字，不会进行换行。
> 
> Console.ReadLine();中间不输入任意字符，作用为读取一行信息，以按下回车键结束。
> 
> Console.ReadKey();仅仅只读取一个字符，直接结束。

## 变量

### 一、变量类型

#### 1.有符号的整型变量

> sbyte 数据范围：-128~127  占用内存空间：1 （单位：字节，下同）
> 
> int 数据范围：-21亿~21亿 占用内存空间：4
> 
> short 数据范围：-32768~32767 占用内存空间：2
> 
> long 数据范围：-9百万兆~9百万兆 占用内存空间：8

#### 2.无符号的整型变量（内存占用空间同上）

> byte 数据范围：0~255
> 
> uint 数据范围：0~42亿多
> 
> ushort 数据范围：0~65535
> 
> ulong 数据范围：0~18百万兆

#### 3.浮点数（小数）

float  占用内存空间：4

存储7/8位有效数字，根据编译器的不同，有效数字可能也会不同，多出的数字会进行四舍五入。

有效数字：从小数点开始从左往右的个数

注：在声明float变量时，要在小数后面加上 f 来声明这一串小数是float类型的数，因为在c#中小数默认为 double 类型。

```cs
float f = 0.123123123f;
```

double  占用内存空间：8

存储15~17位有效数字。

```cs
double d = 0.123123123123;
```

decimal 占用内存空间：16

存储27~28位有效数字，用途较少，需要在小数后面加上m。

```cs
decimal d = 0.123345123123123123m;
```

#### 4.特殊变量

bool 占用内存空间：1

表示真假的数据类型 true/false

```cs
bool b = true;
b = false;
```

char 占用内存空间：2 （特别注意，有的语言中占用1字节，在C#中占用2字节）

存储单个字符的变量类型

```cs
char c = 'a';
```

string

字符串 类型，存储多个字符，没有上限

```cs
string s = "aaaaaaa";
```

### 二、变量的本质和存储空间（内存中）

我们可以通过sizeof方法，快速查阅变量类型所占用的内存空间（单位：字节）

而变量的本质是什么，变量的本质实际就是一串二进制的数，计算机中所有的数据本质都是二进制，是一堆0和1。

为什么是二进制呢？ 因为数据传递只能通过电信号，只有开和关两种状态。所以就用0和1来表示这两种状态。并且计算机中的存储单位最小为 bit （位），它只能表示0和1两个数字，1bit就是一个数，0/1。为了方便数据表示，出现了一个叫做byte（字节）的单位，它是由8个bit组成的存储单位。

所以我们说，一字节为8位。

### 三、变量的命名规则

#### 必须遵守的规则

> 1.不能重名  
> 2.不能以数字开头  
> 3.不能已程序关键字命名  
> 4.不能有特殊符号（下划线除外）

建议的命名规则：变量名要有含义   用英文（拼音）表示变量的作用  
非常不建议用中文命名。

例：

```cs
string name;
float speed;
```

#### 常用命名规则

驼峰命名法——首字母小写，之后单词首字母大写（变量）

```cs
string myName;
int enemySpeed;
```

帕斯卡命名法——所有单词字母都大写（函数、类）

```cs
private void CheckForDistance(){}
```

## 类型转换

什么是类型转换？类型转换就是不同变量类型之间的相互转换。

### 一、隐式转换

隐式转换的基本规则：不同类型之间自动转换（大范围装小范围）

#### 1.相同大类型之间的转换

注：decimal不能够隐式转换double类型和float类型，特殊变量的三种也无法隐式转换。

```cs
long ——> int ——> short ——> sbyte
ulong ——> uint ——> ushort ——> byte
double ——> float
long l = 1;
int i = 1;
short s = 1;
sbyte b = 1;
l = i;
```

#### 2.不同大类型之间的转换

##### 无符号与有符号的整型变量

> 无符号的变量是不能够隐式转换装给有符号的变量的
> 
> 而有符号的变量范围必须大于无符号的变量才能够隐式转换装载无符号的变量。
> 
> 即：有符号 = 无符号 （范围够上即可成立） 无符号 = 有符号（不可能成立）

```cs
byte b = 1;
ushort s = 1;
uint i = 1;
ulong l = 1;
sbyte b1 = 1;
short s1 = 1;
int i1 = 1;
long l1 = 1;
b1 = b (不成立)
s1 = b (成立)
s1 = s (不成立)
```

##### 浮点数与整型变量

任何整型变量都可以隐式转换为浮点数，decimal 虽然不能接收 float 和 double 类型的转换，但是可以接收整型变量的转换。

##### 特殊变量

bool 类型不能够隐式存储或者转换任何变量。

char 类型可以隐式转换为整型，实际上为转换为了int类型（即ASCII码）再隐式转换为了别的类型，所以对于存储范围小于int类型的变量类型是转换不了的。

##### 总结

> 隐式转换的规则主要为：
> 
> 高精度（大范围）装载低精度（小范围）
> 
> double ——> float  ——> 整数（无符号、有符号） ——>char
> 
> decimal ——>整数（无符号、有符号） ——>char
> 
> string 和 bool 不参与隐式转换规则

### 二、显式转换（强制转换）

#### 1.括号强转

> 作用：一般情况下，将高精度的类型强制转换为低精度的类型
> 
> 语法： 变量类型 变量名 = （变量类型）变量。
> 
> 需要特别注意精度问题和范围问题。

```cs
有符号整型：
sbyte b = 1;
short s = 1;
int i = 40000;
long l = 1;
括号强转可能会出现范围问题造成的异常
例：
s = (short) i;
输出内容为：-25565
无符号整型同上
而浮点数类型需要注意精度问题：
float f = 1.123123f;
double d = 1.123123123123123;
例：
f = (float) d;
输出内容为：1.12312312。
浮点数和整型转换：
i = (int)f;
输出内容为 1。并且不会进行四舍五入，只会进行舍弃小数点处理。
char和数值类型转换：
i = 'a';
char c = (char)i;
得到的结果是没有问题的。
```

#### 2.Parse方法

> 作用：把字符串类型转换为对应的类型
> 
> 语法：变量类型.Parse("字符串")
> 
> 需要注意的：字符串必须能够转换成对应类型，否则会报错

```cs
int i = int.Parse("123");
float f = float.Parse("1.123123");
bool b = bool.Parse("true");
char c = char.Parse("A");
```

#### 3.Convert方法

> 作用：更准确的将各个类型之间进行转换
> 
> 语法：Convert.To目标类型（变量或者常量）
> 
> 注意：填写的变量或者常量必须正确，否则出错

```cs
int a = Convert.ToInt32("111");
a = Convert.ToInt32(1.123123);
a = Convert.ToInt32(true);
float f = Convert.ToSingle("111");
short s = Convert.ToInt16("111");
long l = Convert.ToInt64("111");
```

## 异常捕获

### 基本语法：

```cs
try
{
  希望进行异常捕获的代码块
  如果出现报错，不会让程序卡死，会转到catch代码块中
}
catch(Exception e)
{
  如果出错会执行此处的代码来捕获异常
  catch(Exception e)为具体的报错跟踪，通过e得到具体的错误信息
  throw;
}
finally
{
  最后执行的代码，不论有没有出现报错都会执行
}
```
