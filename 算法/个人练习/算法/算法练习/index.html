第四关：队列——先进先出

现在开始：递归

递归只有一个核心思想：一个函数在自己的函数体里调用自己。
听起来很绕，但其实你每天都在用这个思维，只是没意识到。
比如”计算 1+2+3+…+n 的总和”，你可以这样想：
sum(n) = n + sum(n-1)
sum(n-1) = (n-1) + sum(n-2)
sum(n-2) = (n-2) + sum(n-3)
…
sum(1) = 1 ← 到这里不再往下，这叫”终止条件”
每一层都把问题缩小一点，直到小到能直接回答为止。
