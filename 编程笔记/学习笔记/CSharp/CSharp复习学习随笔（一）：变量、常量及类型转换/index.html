变量

一、变量类型

1.有符号的整型变量

sbyte 数据范围：-128~127  占用内存空间：1 （单位：字节，下同）

int 数据范围：-21亿~21亿 占用内存空间：4

short 数据范围：-32768~32767 占用内存空间：2

long 数据范围：-9百万兆~9百万兆 占用内存空间：8

2.无符号的整型变量（内存占用空间同上）

byte 数据范围：0~255

uint 数据范围：0~42亿多

ushort 数据范围：0~65535

ulong 数据范围：0~18百万兆

3.浮点数（小数）

float  占用内存空间：4

存储7/8位有效数字，根据编译器的不同，有效数字可能也会不同，多出的数字会进行四舍五入。

有效数字：从小数点开始从左往右的个数

注：在声明float变量时，要在小数后面加上 f 来声明这一串小数是float类型的数，因为在c#中小数默认为 double 类型。

double  占用内存空间：8

存储15~17位有效数字。

decimal 占用内存空间：16

存储27~28位有效数字，用途较少，需要在小数后面加上m。

4.特殊变量

bool 占用内存空间：1

表示真假的数据类型 true/false

char 占用内存空间：2 （特别注意，有的语言中占用1字节，在C#中占用2字节）

存储单个字符的变量类型

string

字符串 类型，存储多个字符，没有上限

二、变量的本质和存储空间（内存中）

我们可以通过sizeof方法，快速查阅变量类型所占用的内存空间（单位：字节）

而变量的本质是什么，变量的本质实际就是一串二进制的数，计算机中所有的数据本质都是二进制，是一堆0和1。

为什么是二进制呢？ 因为数据传递只能通过电信号，只有开和关两种状态。所以就用0和1来表示这两种状态。并且计算机中的存储单位最小为 bit （位），它只能表示0和1两个数字，1bit就是一个数，0/1。为了方便数据表示，出现了一个叫做byte（字节）的单位，它是由8个bit组成的存储单位。

所以我们说，一字节为8位。

建议的命名规则：变量名要有含义   用英文（拼音）表示变量的作用
非常不建议用中文命名。

类型转换

什么是类型转换？类型转换就是不同变量类型之间的相互转换。

一、隐式转换

隐式转换的基本规则：不同类型之间自动转换（大范围装小范围）

1.相同大类型之间的转换

注：decimal不能够隐式转换double类型和float类型，特殊变量的三种也无法隐式转换。

2.不同大类型之间的转换

无符号与有符号的整型变量

无符号的变量是不能够隐式转换装给有符号的变量的

而有符号的变量范围必须大于无符号的变量才能够隐式转换装载无符号的变量。

即：有符号 = 无符号 （范围够上即可成立） 无符号 = 有符号（不可能成立）

浮点数与整型变量

任何整型变量都可以隐式转换为浮点数，decimal 虽然不能接收 float 和 double 类型的转换，但是可以接收整型变量的转换。

特殊变量

bool 类型不能够隐式存储或者转换任何变量。

char 类型可以隐式转换为整型，实际上为转换为了int类型（即ASCII码）再隐式转换为了别的类型，所以对于存储范围小于int类型的变量类型是转换不了的。

总结

隐式转换的规则主要为：

高精度（大范围）装载低精度（小范围）

double ——> float  ——> 整数（无符号、有符号） ——>char

decimal ——>整数（无符号、有符号） ——>char

string 和 bool 不参与隐式转换规则
